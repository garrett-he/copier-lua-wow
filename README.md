A [Copier][1] template for [World of Warcraft][2] addons.

## Quickstart

1. Install the latest Copier via command:

    ```text
    pip install --upgrade copier copier-template-extensions
    ```

2. Use command `copier` to generate a new project:

    ```text
    copier copy gh:garrett-he/copier-lua-wow /path/to/destination
    ```
